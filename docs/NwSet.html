Class NwSet

A normalization and weighting set.

~

Properties:

weightedScoreUnit string This is the optional unit of the (normalized and) weighted score when this normalization and weighting set was applied on a LCIA result.
factors List[NwFactor] The list of normalization and weighting factors of this set.
