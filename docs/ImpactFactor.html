Class ImpactFactor

A single characterisation factor of a LCIA category for a flow.

~

Properties:

flow Ref[Flow] The Flow of the impact assessment factor.
location Ref[Location] In case of a regionalized impact category, this field can contain the location for which this factor is valid.
flowProperty Ref[FlowProperty] The quantity of the flow to which the LCIA factor is related (e.g. Mass).
unit Ref[Unit] The flow unit to which the LCIA factor is related (e.g. kg).
value double The value of the impact assessment factor.
formula string A mathematical formula for calculating the value of the LCIA factor.
uncertainty Uncertainty The uncertainty distribution of the factors' value.

~
