Class DQScore

An score value of an indicator (DQIndicator) in a data quality system (DQSystem).

~

Properties:

position int
label string
description string
uncertainty double

~

JSON-LD Example

{
    "@type": "DQScore",
    "position": 5,
    "description": "Data from unknown or distinctly different area (North America instead of Middle East, OECD-Europe instead of Russia)",
    "uncertainty": 1.1
}
