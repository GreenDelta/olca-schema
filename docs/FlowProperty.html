Class FlowProperty

A flow property is a quantity that can be used to express amounts of a flow.

~

Properties:

flowPropertyType FlowPropertyType The type of the flow property
unitGroup Ref[UnitGroup] The units of measure that can be used to express quantities of the flow property.

~

library string If this entity is part of a library, this field contains the identifier of that library. The identifier is typically just the combination of the library name and version.
