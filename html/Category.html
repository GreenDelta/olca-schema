Class Category

A category for the categorisation of types like processes, flows, etc.

~

Properties:

childCategories List[Category] A list of categories that are sub-categories of the category.
modelType ModelType The type of models that can be linked to the category.
parentCategory Category The parent category of the category.

~

Properties from RootEntity:

name string The name of the entity.
description string The description of the entity.
version string A version number in ILCD format (e.g. 1.0 or 1.0.1).
lastChange dateTime The date when the entity was changed the last time. Together with the UUID and version this identifies an entity unambiguously.
