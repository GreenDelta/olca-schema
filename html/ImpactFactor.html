Class ImpactFactor

A single characterisation factor of a LCIA category for a flow.

~

Properties:

flow Flow The flow of the impact assessment factor.
flowProperty FlowProperty The quantity of the flow to which the LCIA factor is related (e.g. Mass).
unit Unit The flow unit to which the LCIA factor is related (e.g. kg).
value double The value of the impact assessment factor.
formula string A mathematical formula for calculating the value of the LCIA factor.
uncertainty Uncertainty The uncertainty distribution of the factors' value.

~
