Class ProductSystem

A product system describes the supply chain of a product (the functional unit) ...

~

Properties:

processes List[Process] The descriptors of all processes that are contained in the product system.
referenceProcess Process The descriptor of the process that provides the flow of the functional unit of the product system.
referenceExchange Exchange The exchange of the reference processes (typically the product output) that provides the flow of the functional unit of the product system.
targetAmount double The flow amount of the functional unit of the product system.
targetUnit Unit The unit in which the flow amount of the functional unit is given.
targetFlowProperty FlowProperty The flow property in which the flow amount of the functional unit is given.
processLinks List[ProcessLink] The process links of the product system.

~

Properties from CategorizedEntity:

category Category The category of the entity.

~

Properties from RootEntity:

name string The name of the entity.
description string The description of the entity.
version string A version number in MAJOR.MINOR.PATCH format where the MINOR and PATCH fields are optional and the fields may have leading zeros (so 01.00.00 is the same as 1.0.0 or 1).
lastChange dateTime The timestamp when the entity was changed the last time.

~
