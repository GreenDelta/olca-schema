Class Source

A source is a literature reference.

~

Properties:

doi string The digital object identifier of the source (see http://en.wikipedia.org/wiki/Digital_object_identifier).
textReference string The full text reference of the source.
year integer The publication year of the source.
externalFile string A direct link (relative or absolute URL) to the source file.

~

Properties from CategorizedEntity:

category Category The category of the entity.

~

Properties from RootEntity:

name string The name of the entity.
description string The description of the entity.
version string A version number in ILCD format (e.g. 1.0 or 1.0.1).
lastChange dateTime The date when the entity was changed the last time. Together with the UUID and version this identifies an entity unambiguously.
