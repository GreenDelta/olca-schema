Class Exchange

~

flow Flow The flow of the exchange.
flowProperty FlowProperty The quantity in which the amount is given.
input boolean
quantitativeReference boolean Indicates whether the exchange is the quantitative reference of the process.
baseUncertainty double
provider Process
amount double
amountFormula string
unit Unit
pedigreeUncertainty string
uncertainty Uncertainty
comment string A general comment about the input or output.
