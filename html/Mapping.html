Class Mapping

A mapping from one flow to another.

~

Properties:

from List[Ref[Flow],Ref[FlowProperty],Ref[Unit]] The flow, flow property, and unit of the source flow.
to List[Ref[Flow],Ref[FlowProperty],Ref[Unit]] The flow, flow property, and unit of the target flow.
conversionFactor double The factor to convert the original source flow to the target flow.

~

Properties from RootEntity:

name string The name of the entity.
description string The description of the entity.
version string A version number in MAJOR.MINOR.PATCH format where the MINOR and PATCH fields are optional and the fields may have leading zeros (so 01.00.00 is the same as 1.0.0 or 1).
lastChange dateTime The timestamp when the entity was changed the last time.
